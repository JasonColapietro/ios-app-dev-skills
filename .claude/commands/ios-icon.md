# /ios-icon

Input: `$ARGUMENTS` - app name, brand assets, site URL, or existing project.

Use the `ios-swiftui-product` and `ios-aso-launch` skills.

Create a practical icon and launch-screen spec:

- App Store icon concept and visual hierarchy;
- source asset requirements;
- iOS icon slots and export checklist;
- launch screen behavior;
- light/dark fit where relevant;
- avoid text-heavy or misleading icons.

Output icon direction, required files, export sizes, and implementation path.
