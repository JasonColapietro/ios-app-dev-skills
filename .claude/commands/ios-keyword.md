# /ios-keyword

Input: `$ARGUMENTS` - app idea or seed keyword.

Use the `ios-aso-launch` skill.

Create `aso/keywords.md` with:

- target keyword,
- secondary keywords,
- competitor scan,
- avoid list,
- sub-niche positioning,
- notes for screenshots and metadata.

Gate the output against App Store keyword rules: no competitor trademarks, no
blind category terms, and a clear reason the target keyword is winnable.
